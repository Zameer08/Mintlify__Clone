# Mintlify Landing Page Clone (Desktop)

A **desktop-only HTML & CSS recreation of the Mintlify website landing page**, focused on **layout accuracy, typography, spacing, and visual hierarchy**.  
The goal was to closely match the original design, not to add interactivity or creativity.

---

## Sections Recreated

- Navigation Bar
- Hero Section
- Trusted By / Logos
- Feature Sections
- Feature Cards
- Testimonials
- Use Cases / Stories
- Changelog / Updates
- Team / About
- Final CTA
- Footer

---

## Tech & Constraints

- HTML5, CSS3
- No JavaScript
- No TailwindCSS
- No animations
- Desktop-only (not responsive)

---

## Fonts & Colors

- **Font:** Inter (with system fallbacks)
- light theme and accent CTA color
